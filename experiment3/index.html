Experiment 3 - Alternate Worlds

Description

This experiment is attempting to use autotiling logic to generate a randomized map.

Technical

The map grid is created using an array of coordinates, generated in the function generateGrid(). The map is then drawn in drawGrid(), by looking at each coordinate of the grid and placing an image corresponding with the value. The grids are randomized with noise and the movement of water is also generated with noise. gridContext() is then used to ensure differences in the tiles.
